# 發票處理系統 (Invoice Processor)

## 功能說明
本工具提供發票驗證和支出分類功能，用於自動化處理企業支出記錄。

### 核心功能
- **validate_invoice()**: 驗證發票必需欄位（日期、金額、統編）
- **categorize_expense()**: 根據金額自動分類支出類別

### 分類規則
| 金額範圍 | 分類 | 備註 |
|--------|------|------|
| < NT$1,000 | 日常雜支 | 自動核准 |
| NT$1,000 ~ 5,000 | 辦公用品 | 自動核准 |
| > NT$5,000 | 固定資產 | 需主管審核 |
| > NT$10,000 | 高額支出 | 【高額警報】 |
