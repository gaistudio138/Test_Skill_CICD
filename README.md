# Invoice Validator & Categorizer (V3)

這個專案提供了一個簡單的 Python 工具，用於驗證發票資料並根據金額自動分類支出。

## 🚀 特點

*   **發票驗證**: 檢查發票資料是否包含必要的欄位 (`date`, `amount`, `tax_id`)。
*   **支出分類**: 根據金額自動將支出分類為「日常雜支」、「辦公用品」或「固定資產」。
*   **高額警報**: 當支出金額超過 NT$10,000 時，發出警報，提示需要主管審核。

## 🛠️ 使用方法

您可以直接執行 Python 腳本，或將其整合到您的應用程式中。

### 程式碼說明

*   `validate_invoice(invoice_data)`:
    *   **功能**: 接收一個字典形式的發票資料，檢查是否包含 `"date"`, `"amount"`, `"tax_id"` 這三個必要欄位。
    *   **回傳**:
        *   若缺少欄位，回傳 `{"status": "error", "message": "缺少欄位: [缺少的欄位列表]"}`。
        *   若所有欄位都存在，回傳 `{"status": "success", "message": "發票格式正確"}`。

*   `categorize_expense(amount)`:
    *   **功能**: 根據提供的金額 (`amount`)，將支出進行分類。
    *   **分類規則**:
        *   小於 1000: `"日常雜支"`
        *   1000 到 5000 (包含 5000): `"辦公用品"`
        *   大於 5000: `"固定資產"`
    *   **回傳**: 分類名稱的字串。

### 範例執行

當您執行主程式 (`if __name__ == "__main__":`) 時，會執行以下測試：

```python
# 模擬測試資料
test_data = {"date": "2026-04-17", "amount": 100, "tax_id": "12345678"}

print("--- 發票驗證測試V3 ---")
result = validate_invoice(test_data)
print(f"結果: {result['message']}")

category = categorize_expense(test_data['amount'])
print(f"分類: {category}")

# 高額警報檢查 (金額 > 10000)
if test_data['amount'] > 10000:
    print("【高額警報】此筆支出超過 NT$10,000，需主管審核")
```

執行上述腳本，預期輸出為：

```
--- 發票驗證測試V3 ---
結果: 發票格式正確
分類: 日常雜支
```
